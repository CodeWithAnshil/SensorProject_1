About Sensor Fault Detection

Sensor Fault Detection system is a Machine Learning based application designed to identify faulty sensors in wafer manufacturing processes.

Problem Statement

In semiconductor manufacturing, faulty sensors can produce incorrect readings which may lead to production defects and financial loss.

Solution Approach

This system uses historical sensor data in CSV format and applies trained Machine Learning models to classify sensors as Good or Faulty.
